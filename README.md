# Breast Cancer Prediction App
The models were developed and evaluated in the following order: Logistic Regression, ANN, and XGBoost. This sequence is reflected in the notebook and can be followed step by step.

After comparing model performance, XGBoost was selected as the final model due to its overall effectiveness. The deployed application also uses the XGBoost model for prediction.


This app predicts whether a tumor is benign or malignant using an XGBoost classifier trained on the sklearn breast cancer dataset.
